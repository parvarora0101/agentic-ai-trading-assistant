# agentic-ai-trading-assistant
Autonomous AI Trading Assistant using Agentic AI

This project focuses on building an autonomous AI-based trading assistant
using agentic artificial intelligence concepts.

## Problem Statement
Traditional trading bots rely on fixed strategies and fail to adapt to
dynamic market conditions.

## Objectives
- Analyze market data autonomously
- Make buy/sell/hold decisions
- Manage risk dynamically
- Learn from previous trades

## Tech Stack
- Python
- Pandas, NumPy
- Alpaca / Zerodha API
